## 📋 Description

Intensive **14-week (~3.5 months)** bootcamp focused on mastering **modern Python** from scratch. Designed to take students from absolute beginners to **Junior Python Developer**, with emphasis on clean code, best practices, and real-world projects.

### 🎯 Objectives

Upon completing the bootcamp, students will be able to:

- ✅ Master the fundamentals of modern Python (3.13+)
- ✅ Apply type hints and static typing correctly
- ✅ Work with complex data structures
- ✅ Implement Object-Oriented Programming (OOP)
- ✅ Handle files, exceptions, and modules
- ✅ Write clean and maintainable code
- ✅ Create automated tests with pytest
- ✅ Use version control with Git
- ✅ Work with popular Python ecosystem libraries
- ✅ Develop complete projects from scratch

### 🚀 Why Python?

> **Modern Python from day 1** - No legacy code, only current best practices.

Python is the most versatile and in-demand programming language in the world. This bootcamp focuses exclusively on Python 3.13+ with modern features like type hints, match statements, and f-strings. Students learn directly the tools and techniques they will use in the professional world.

## 🗓️ Bootcamp Structure

|        Stage         | Weeks | Hours | Main Topics                                        |
| :------------------: | :---: | :---: | -------------------------------------------------- |
|   **Fundamentals**   |  1-4  |  24h  | Variables, types, conditionals, loops, functions   |
| **Data Structures**  |  5-7  |  18h  | Lists, dictionaries, sets, tuples, algorithms      |
| **OOP & Modularity** | 8-10  |  18h  | Classes, inheritance, modules, packages, uv        |
| **Advanced Topics**  | 11-13 |  18h  | Files, exceptions, decorators, testing             |
|  **Final Project**   |  14   |  6h   | Capstone project, external libraries, presentation |

**Total: 14 weeks** | **84 hours** of intensive training

## 🚀 Quick Start

### Prerequisites

- **Docker** and **Docker Compose** installed
- **Git** for version control
- **VS Code** (recommended) with included extensions
- Modern browser (Chrome, Firefox, Edge)

### 1. Clone the Repository

```bash
git clone https://github.com/epti-dev/bc-python.git
cd bc-python
```

### 2. Install VS Code Extensions

```bash
# Open in VS Code
code .

# Recommended extensions will appear automatically
# Or run: Ctrl+Shift+P → "Extensions: Show Recommended Extensions"
```

### 3. Navigate to Current Week

```bash
cd bootcamp/week-01
```

### 4. Follow Instructions

Each week contains a `README.md` with detailed instructions.

## 📊 Learning Methodology

### Teaching Strategies

- 🎯 **Project-Based Learning (PBL)**
- 🧩 **Deliberate Practice**
- 🔄 **Coding Challenges**
- 👥 **Peer Code Review**
- 🎮 **Live Coding**

### Time Distribution (6h/week)

- **Theory**: 1.5-2 hours
- **Exercises**: 2.5-3 hours
- **Project**: 1.5-2 hours

### Assessment

Each week includes three types of evidence:

1. **Knowledge 🧠** (30%): Quizzes and theoretical assessments
2. **Performance 💪** (40%): Practical in-class exercises
3. **Product 📦** (30%): Evaluable deliverables (functional projects)

**Passing criteria**: Minimum 70% in each type of evidence
