# 📚 eBooks Gratuitos - Semana 01

## Libros Recomendados

### 1. Automate the Boring Stuff with Python
- **Autor**: Al Sweigart
- **Idioma**: Inglés
- **URL**: https://automatetheboringstuff.com/
- **Capítulos relevantes**: 1-2 (Basics, Flow Control)
- **Por qué leerlo**: Enfoque práctico, ejemplos del mundo real

### 2. Python Crash Course (Extractos)
- **Autor**: Eric Matthes
- **Idioma**: Inglés
- **URL**: https://ehmatthes.github.io/pcc/
- **Capítulos relevantes**: 1-2 (Getting Started, Variables)
- **Por qué leerlo**: Excelente para principiantes absolutos

### 3. Think Python (2da Edición)
- **Autor**: Allen B. Downey
- **Idioma**: Inglés (hay traducción española)
- **URL**: https://greenteapress.com/thinkpython2/
- **Capítulos relevantes**: 1-2 (The Way of the Program, Variables)
- **Por qué leerlo**: Enfoque en pensamiento computacional

### 4. A Byte of Python
- **Autor**: Swaroop C H
- **Idioma**: Español disponible
- **URL**: https://python.swaroopch.com/
- **Capítulos relevantes**: Basics, Operators and Expressions
- **Por qué leerlo**: Conciso y directo al grano

---

## 📖 Cómo usar estos recursos

1. **No intentes leer todo** - Enfócate en los capítulos relevantes para esta semana
2. **Practica mientras lees** - Ejecuta los ejemplos en tu entorno Docker
3. **Toma notas** - Escribe tus propios ejemplos basados en lo que lees
4. **Complementa, no reemplaces** - Estos recursos son adicionales al material del bootcamp

---

## 💡 Tip

Si el inglés no es tu fuerte, "A Byte of Python" tiene traducción al español y es excelente para comenzar.
