## 🎯 Cómo aprovechar los videos

### Antes de ver
- [ ] Ten tu entorno Docker listo
- [ ] Prepara un archivo Python para practicar

### Mientras ves
- [ ] Pausa y escribe el código tú mismo
- [ ] No copies, intenta entender y escribir
- [ ] Experimenta cambiando valores

### Después de ver
- [ ] Resume los conceptos clave
- [ ] Intenta explicar lo aprendido en voz alta
- [ ] Aplica lo aprendido en los ejercicios del bootcamp
