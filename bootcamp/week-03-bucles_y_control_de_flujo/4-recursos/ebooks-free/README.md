# 📚 Libros Electrónicos Gratuitos - Semana 3

## Bucles y Control de Flujo en Python

### 1. Automate the Boring Stuff with Python (Capítulo 2)
- **Autor**: Al Sweigart
- **Enlace**: https://automatetheboringstuff.com/2e/chapter2/
- **Idioma**: Inglés
- **Descripción**: Capítulo dedicado a control de flujo, incluyendo bucles for y while con ejemplos prácticos de automatización.

### 2. Python for Everybody (Capítulos 4-5)
- **Autor**: Charles Severance
- **Enlace**: https://www.py4e.com/html3/
- **Idioma**: Inglés (con versión en español disponible)
- **Descripción**: Capítulos sobre funciones e iteraciones con enfoque en análisis de datos.

### 3. Think Python (Capítulo 7)
- **Autor**: Allen B. Downey
- **Enlace**: https://greenteapress.com/thinkpython2/html/thinkpython2008.html
- **Idioma**: Inglés
- **Descripción**: Capítulo sobre iteración que explica while, break y algoritmos iterativos.

### 4. A Byte of Python
- **Autor**: Swaroop C H
- **Enlace**: https://python.swaroopch.com/control_flow.html
- **Idioma**: Inglés (traducciones disponibles)
- **Descripción**: Sección de control de flujo clara y concisa, ideal para principiantes.

Para esta semana, recomendamos especialmente:
