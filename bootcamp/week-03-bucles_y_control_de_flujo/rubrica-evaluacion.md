# 📊 Rúbrica de Evaluación - Semana 03

## 🎯 Competencias a Evaluar

Esta semana se evalúan las siguientes competencias:

1. **Bucles for**: Uso correcto de iteración sobre secuencias
2. **Función range()**: Generación de secuencias numéricas
3. **Bucles while**: Repetición condicional y validación
4. **Control de flujo**: Uso de break, continue y else
5. **Patrones de iteración**: Aplicación de patrones comunes

## 🔄 Retroalimentación

### Fortalezas Comunes
- Buen uso de `for` para iteración básica
- Comprensión de `range()` con un parámetro
- Código funcional y legible

### Áreas de Mejora Frecuentes
- Confusión entre `range(n)` y `range(1, n+1)`
- Bucles infinitos por olvidar incrementar variable
- No usar `break` cuando es apropiado
- Modificar lista mientras se itera
