# 📚 E-books Gratuitos - Semana 10

## Clases Abstractas, Módulos y Paquetes

### 📖 Libros Recomendados

#### 1. Python Official Documentation
- **Título**: The Python Tutorial - Modules
- **Autor**: Python Software Foundation
- **URL**: https://docs.python.org/3/tutorial/modules.html
- **Descripción**: Documentación oficial sobre módulos y paquetes en Python
- **Nivel**: Intermedio

#### 2. Dive Into Python 3
- **Título**: Dive Into Python 3 - Chapter 6: Classes & Iterators
- **Autor**: Mark Pilgrim
- **URL**: https://diveintopython3.net/
- **Descripción**: Capítulo sobre clases avanzadas y patrones
- **Nivel**: Intermedio

#### 3. Python Packaging User Guide
- **Título**: Python Packaging User Guide
- **Autor**: PyPA (Python Packaging Authority)
- **URL**: https://packaging.python.org/
- **Descripción**: Guía completa sobre empaquetado y distribución
- **Nivel**: Intermedio-Avanzado

#### 4. Architecture Patterns with Python
- **Título**: Architecture Patterns with Python
- **Autores**: Harry Percival & Bob Gregory
- **URL**: https://www.cosmicpython.com/
- **Descripción**: Patrones de arquitectura incluyendo inyección de dependencias
- **Nivel**: Avanzado
- **Nota**: Libro completo disponible gratis online
