### Quick Reference
```
dict()              # Crear diccionario vacío
dict.fromkeys()     # Crear con claves predefinidas
d[key]              # Acceder (KeyError si no existe)
d.get(key)          # Acceder seguro (None si no existe)
d[key] = value      # Agregar/actualizar
del d[key]          # Eliminar
d.pop(key)          # Eliminar y retornar
d.keys()            # Vista de claves
d.values()          # Vista de valores
d.items()           # Vista de pares
d.update(other)     # Combinar diccionarios
d | other           # Merge (Python 3.9+)
d.copy()            # Copia superficial
d.clear()           # Vaciar diccionario
```
