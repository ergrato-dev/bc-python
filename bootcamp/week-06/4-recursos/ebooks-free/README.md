# 📚 Libros Electrónicos Gratuitos - Semana 6

## Diccionarios en Python

### 1. Python Official Tutorial - Data Structures
- **Enlace**: https://docs.python.org/3/tutorial/datastructures.html#dictionaries
- **Descripción**: Documentación oficial de Python sobre diccionarios
- **Nivel**: Principiante a Intermedio
- **Idioma**: Inglés

### 2. Think Python (2nd Edition)
- **Autor**: Allen B. Downey
- **Enlace**: https://greenteapress.com/wp/think-python-2e/
- **Capítulo relevante**: Chapter 11 - Dictionaries
- **Descripción**: Excelente introducción a diccionarios con ejercicios prácticos
- **Nivel**: Principiante
- **Idioma**: Inglés (disponible traducción al español)

### 3. Automate the Boring Stuff with Python
- **Autor**: Al Sweigart
- **Enlace**: https://automatetheboringstuff.com/
- **Capítulo relevante**: Chapter 5 - Dictionaries and Structuring Data
- **Descripción**: Enfoque práctico con ejemplos del mundo real
- **Nivel**: Principiante
- **Idioma**: Inglés

### 4. Python for Everybody
- **Autor**: Charles Severance
- **Enlace**: https://www.py4e.com/book
- **Capítulo relevante**: Chapter 9 - Dictionaries
- **Descripción**: Libro introductorio muy accesible
- **Nivel**: Principiante
- **Idioma**: Inglés (disponible en español)

### 5. A Byte of Python
- **Autor**: Swaroop C H
- **Enlace**: https://python.swaroopch.com/
- **Capítulo relevante**: Data Structures
- **Descripción**: Guía concisa y práctica
- **Nivel**: Principiante
- **Idioma**: Inglés (múltiples traducciones disponibles)
