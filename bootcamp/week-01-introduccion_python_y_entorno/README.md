## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Entender qué es Python y por qué es tan popular
- ✅ Configurar tu entorno de desarrollo con Docker y VS Code
- ✅ Escribir y ejecutar tu primer programa en Python
- ✅ Trabajar con variables y tipos de datos básicos
- ✅ Usar operadores aritméticos y de comparación
- ✅ Aplicar type hints desde el primer día

---

## 📚 Requisitos Previos

- Computadora con Windows, macOS o Linux
- Docker Desktop instalado
- VS Code instalado
- Ganas de aprender 🚀

## 💡 Tips para esta Semana

> 🎯 **Consejo #1**: No te preocupes si al principio todo parece nuevo. Python está diseñado para ser fácil de leer y entender.

> 🐍 **Consejo #2**: Escribe código todos los días, aunque sean solo 15 minutos. La práctica constante es clave.

> 🔧 **Consejo #3**: Si algo no funciona, lee el mensaje de error con calma. Python te dice exactamente qué está mal.
