## 📚 Conceptos Cubiertos

### Métodos para Agregar
- `append(item)` - Agrega UN elemento al final
- `extend(iterable)` - Agrega MÚLTIPLES elementos
- `insert(index, item)` - Inserta en posición específica

### Métodos para Eliminar
- `remove(item)` - Elimina primera ocurrencia por valor
- `pop([index])` - Elimina y retorna por índice
- `clear()` - Vacía la lista

### Métodos de Búsqueda
- `index(item)` - Encuentra posición
- `count(item)` - Cuenta ocurrencias

### Métodos de Ordenamiento
- `sort()` - Ordena in-place
- `reverse()` - Invierte in-place

### Métodos de Copia
- `copy()` - Crea copia superficial

| `remove()` | ✅ Sí | `None` |
| `pop()` | ✅ Sí | Elemento |
| `sort()` | ✅ Sí | `None` |
| `reverse()` | ✅ Sí | `None` |
| `index()` | ❌ No | `int` |
| `count()` | ❌ No | `int` |
