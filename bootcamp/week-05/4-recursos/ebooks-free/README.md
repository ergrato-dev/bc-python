# 📚 eBooks Gratuitos - Semana 05

## Listas y Tuplas en Python

### 1. Python Documentation - Data Structures
- **Enlace**: [docs.python.org/3/tutorial/datastructures.html](https://docs.python.org/3/tutorial/datastructures.html)
- **Idioma**: Inglés
- **Descripción**: Documentación oficial sobre estructuras de datos en Python
- **Temas**: Listas, tuplas, diccionarios, sets, técnicas de looping

### 2. Think Python - Chapter 10: Lists
- **Enlace**: [greenteapress.com/thinkpython2](https://greenteapress.com/thinkpython2/html/thinkpython2011.html)
- **Autor**: Allen B. Downey
- **Idioma**: Inglés
- **Descripción**: Capítulo completo sobre listas con ejercicios
- **Nivel**: Principiante

### 3. Think Python - Chapter 12: Tuples
- **Enlace**: [greenteapress.com/thinkpython2](https://greenteapress.com/thinkpython2/html/thinkpython2013.html)
- **Autor**: Allen B. Downey
- **Idioma**: Inglés
- **Descripción**: Capítulo dedicado a tuplas y su uso
- **Nivel**: Principiante

### 4. Automate the Boring Stuff - Chapter 4: Lists
- **Enlace**: [automatetheboringstuff.com](https://automatetheboringstuff.com/2e/chapter4/)
- **Autor**: Al Sweigart
- **Idioma**: Inglés
- **Descripción**: Introducción práctica a listas con ejemplos del mundo real
- **Nivel**: Principiante

### 5. Python 101 - Lists, Tuples and Dictionaries
- **Enlace**: [python101.pythonlibrary.org](https://python101.pythonlibrary.org/)
- **Autor**: Michael Driscoll
- **Idioma**: Inglés
- **Descripción**: Guía completa de estructuras de datos básicas
- **Nivel**: Principiante-Intermedio
