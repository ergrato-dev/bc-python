## 🎯 Objetivo

Dominar el uso de tuplas, tuple unpacking y estructuras de datos anidadas en Python.

## 📚 Conceptos Cubiertos

### Tuplas
- Crear tuplas con `()` y sin paréntesis
- Inmutabilidad
- Métodos `count()` e `index()`

### Tuple Unpacking
- Unpacking básico
- Extended unpacking con `*`
- Ignorar valores con `_`
- Swap de variables

### Named Tuples
- `typing.NamedTuple`
- Acceso por nombre y por índice

### Estructuras Anidadas
- Listas de listas (matrices)
- Listas de tuplas
- Acceso y modificación

---

## 🔑 Comparación Tupla vs Lista

| Característica | Tupla | Lista |
|----------------|-------|-------|
| Sintaxis | `(1, 2, 3)` | `[1, 2, 3]` |
| Mutable | ❌ No | ✅ Sí |
| Hasheable | ✅ Sí* | ❌ No |
| Clave de dict | ✅ Sí* | ❌ No |
| Performance | Más rápida | Más lenta |
| Métodos | 2 | 11+ |

*Solo si todos sus elementos son hasheables
