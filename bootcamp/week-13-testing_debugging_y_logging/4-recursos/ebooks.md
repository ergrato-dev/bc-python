# 📚 eBooks Gratuitos - Semana 13

## Testing con pytest

### Python Testing with pytest (fragmentos gratuitos)

- **Autor**: Brian Okken
- **Enlace**: [Python Testing with pytest](https://pragprog.com/titles/bopytest2/python-testing-with-pytest-second-edition/)
- **Descripción**: El libro de referencia para pytest. Cubre desde lo básico hasta fixtures avanzadas, plugins y CI/CD.
- **Contenido relevante**: Capítulo 1 y fragmentos disponibles gratuitamente.

### Test-Driven Development with Python (gratuito)

- **Autor**: Harry J.W. Percival
- **Enlace**: [Obey the Testing Goat!](https://www.obeythetestinggoat.com/)
- **Descripción**: Libro completo sobre TDD con Python y Django. Disponible gratuitamente en línea.
- **Contenido relevante**: Capítulos 1-4 sobre fundamentos de testing.

## Debugging y Logging

### Python Debugging Handbook (Replit)

- **Enlace**: [Python Debugging Handbook](https://docs.replit.com/tutorials/python-debugging)
- **Descripción**: Guía práctica sobre técnicas de debugging en Python.
- **Contenido relevante**: Uso de pdb, breakpoints, estrategias de debugging.

### Real Python - Python Debugging (artículo extenso)

- **Enlace**: [Python Debugging](https://realpython.com/python-debugging-pdb/)
- **Descripción**: Tutorial completo sobre pdb y técnicas de debugging.
- **Formato**: Artículo largo con ejemplos detallados, equivalente a un capítulo de libro.
