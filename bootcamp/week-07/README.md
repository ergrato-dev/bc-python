# Semana 7: Sets y Algoritmos Básicos 🔢

## 📋 Descripción

Esta semana cierra el módulo de **Estructuras de Datos** explorando los **sets** (conjuntos) y sus poderosas operaciones matemáticas. También introduciremos algoritmos básicos de búsqueda y ordenamiento que son fundamentales en programación.

![Week 07 Header](0-assets/week-07-header.svg)

---

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Crear y manipular sets en Python
- ✅ Aplicar operaciones de conjuntos (unión, intersección, diferencia)
- ✅ Entender cuándo usar sets vs listas vs diccionarios
- ✅ Eliminar duplicados eficientemente
- ✅ Implementar búsqueda lineal y binaria
- ✅ Comprender algoritmos de ordenamiento básicos
- ✅ Analizar complejidad temporal (Big O básico)

## 🎯 Proyecto de la Semana

### Analizador de Datos

Crearás un sistema que utiliza sets para:

- Encontrar elementos únicos en colecciones
- Comparar conjuntos de datos (usuarios, productos, etc.)
- Implementar búsqueda eficiente
- Ordenar resultados con diferentes criterios

**Funcionalidades:**
- Análisis de usuarios activos vs registrados
- Detección de duplicados
- Búsqueda de elementos comunes entre grupos
- Ordenamiento por múltiples criterios
