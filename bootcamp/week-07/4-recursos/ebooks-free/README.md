# Recursos: eBooks Gratuitos

## 📚 Libros Recomendados para Sets y Algoritmos

### 1. Think Python (3rd Edition)
- **Autor**: Allen B. Downey
- **Enlace**: https://greenteapress.com/wp/think-python-3rd-edition/
- **Capítulos relevantes**:
  - Chapter 11: Dictionaries and Sets
  - Chapter 12: Tuples and Sets
- **Idioma**: Inglés
- **Por qué leerlo**: Excelente introducción a estructuras de datos con ejemplos claros

### 2. Python Data Structures and Algorithms
- **Autor**: Benjamin Baka
- **Enlace**: Disponible en bibliotecas digitales
- **Capítulos relevantes**:
  - Chapter 3: Algorithm Design Techniques
  - Chapter 5: Searching
  - Chapter 6: Sorting
- **Idioma**: Inglés
- **Por qué leerlo**: Profundiza en algoritmos de búsqueda y ordenamiento

### 3. Problem Solving with Algorithms and Data Structures using Python
- **Autores**: Brad Miller, David Ranum
- **Enlace**: https://runestone.academy/ns/books/published/pythonds/index.html
- **Capítulos relevantes**:
  - Chapter 5: Searching and Sorting
  - Chapter 6: Trees and Tree Algorithms
- **Idioma**: Inglés
- **Por qué leerlo**: Libro interactivo con visualizaciones de algoritmos

### 4. Automate the Boring Stuff with Python (2nd Edition)
- **Autor**: Al Sweigart
- **Enlace**: https://automatetheboringstuff.com/
- **Capítulos relevantes**:
  - Chapter 5: Dictionaries and Structuring Data
- **Idioma**: Inglés
- **Por qué leerlo**: Enfoque práctico con ejemplos del mundo real

### 5. A Common-Sense Guide to Data Structures and Algorithms
- **Autor**: Jay Wengrow
- **Enlace**: Disponible en bibliotecas digitales
- **Capítulos relevantes**:
  - Chapter 1-2: Big O Notation
  - Chapter 9: Recursion
  - Chapter 11: Node-Based Data Structures
- **Idioma**: Inglés
- **Por qué leerlo**: Explicaciones muy claras de complejidad algorítmica
