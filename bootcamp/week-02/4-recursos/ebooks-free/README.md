Recursos bibliográficos gratuitos relacionados con condicionales y control de flujo en Python.

## 📖 Libros Completos

### 1. Automate the Boring Stuff with Python
- **Autor**: Al Sweigart
- **Capítulo relevante**: Chapter 2 - Flow Control
- **Enlace**: [automatetheboringstuff.com](https://automatetheboringstuff.com/2e/chapter2/)
- **Idioma**: Inglés
- **Descripción**: Excelente introducción a if/elif/else con ejemplos prácticos del mundo real.

### 2. Think Python 3rd Edition
- **Autor**: Allen B. Downey
- **Capítulo relevante**: Chapter 5 - Conditionals and recursion
- **Enlace**: [greenteapress.com/thinkpython](https://greenteapress.com/thinkpython/thinkpython.pdf)
- **Idioma**: Inglés
- **Descripción**: Enfoque académico pero accesible a las estructuras de control.

### 3. Python para Todos
- **Autor**: Charles Severance
- **Capítulo relevante**: Capítulo 3 - Ejecución condicional
- **Enlace**: [py4e.com/book](https://www.py4e.com/book)
- **Idioma**: Español disponible
- **Descripción**: Introducción gentil a condicionales con ejercicios progresivos.

### 4. A Byte of Python
- **Autor**: Swaroop C H
- **Capítulo relevante**: Control Flow
- **Enlace**: [python.swaroopch.com](https://python.swaroopch.com/control_flow.html)
- **Idioma**: Inglés (traducciones disponibles)
- **Descripción**: Libro conciso ideal para principiantes.
