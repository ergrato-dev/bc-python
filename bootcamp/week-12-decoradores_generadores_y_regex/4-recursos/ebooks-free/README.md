# 📚 Libros Electrónicos Gratuitos - Semana 12

## Decoradores, Generadores y Expresiones Regulares

### 1. Python Cookbook (Capítulos relevantes)

- **Autor**: David Beazley, Brian K. Jones
- **Capítulos**: 7 (Functions), 8 (Classes), 4 (Iterators and Generators)
- **URL**: Disponible parcialmente en [O'Reilly](https://www.oreilly.com/library/view/python-cookbook-3rd/9781449357337/)
- **Por qué leerlo**: Recetas prácticas de decoradores y generadores avanzados

### 2. Fluent Python (Capítulos de muestra)

- **Autor**: Luciano Ramalho
- **Capítulos gratuitos**: Chapter 7 (Decorators and Closures), Chapter 17 (Iterators)
- **URL**: [Muestra gratuita](https://www.oreilly.com/library/view/fluent-python-2nd/9781492056348/)
- **Por qué leerlo**: La mejor explicación de closures y decoradores

### 3. Regular Expressions Cookbook

- **Autor**: Jan Goyvaerts, Steven Levithan
- **URL**: [Muestra en O'Reilly](https://www.oreilly.com/library/view/regular-expressions-cookbook/9781449327453/)
- **Por qué leerlo**: Patrones regex para casos de uso reales

### 4. Automate the Boring Stuff with Python (Gratuito)

- **Autor**: Al Sweigart
- **Capítulo relevante**: Chapter 7 - Pattern Matching with Regular Expressions
- **URL**: [https://automatetheboringstuff.com/2e/chapter7/](https://automatetheboringstuff.com/2e/chapter7/)
- **Por qué leerlo**: Introducción práctica a regex con ejemplos cotidianos

### 5. Python 3 Patterns, Recipes and Idioms (Gratuito)

- **Autor**: Bruce Eckel
- **URL**: [https://python-3-patterns-idioms-test.readthedocs.io/](https://python-3-patterns-idioms-test.readthedocs.io/)
- **Secciones relevantes**: Decorators, Generators, Comprehensions
- **Por qué leerlo**: Patrones de diseño con Python moderno
