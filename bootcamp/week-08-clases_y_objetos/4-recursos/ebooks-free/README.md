# 📚 E-Books Gratuitos - Semana 08

## Programación Orientada a Objetos en Python

### 1. Python Official Tutorial - Classes

- **Fuente**: Python.org
- **Enlace**: https://docs.python.org/3/tutorial/classes.html
- **Idioma**: Inglés
- **Descripción**: Tutorial oficial de Python sobre clases y objetos. Cubre desde conceptos básicos hasta herencia múltiple.

### 2. Object-Oriented Programming in Python

- **Autor**: Michael H. Goldwasser, David Letscher
- **Enlace**: https://openbookproject.net/thinkcs/python/english3e/
- **Idioma**: Inglés
- **Descripción**: Capítulos dedicados a POO con ejemplos prácticos y ejercicios.

### 3. A Byte of Python

- **Autor**: Swaroop C H
- **Enlace**: https://python.swaroopch.com/oop.html
- **Idioma**: Inglés (disponible en español)
- **Descripción**: Capítulo sobre POO con explicaciones claras para principiantes.

### 4. Automate the Boring Stuff - Classes

- **Autor**: Al Sweigart
- **Enlace**: https://automatetheboringstuff.com/2e/chapter17/
- **Idioma**: Inglés
- **Descripción**: Enfoque práctico de POO aplicado a automatización.

### 5. Python 101

- **Autor**: Michael Driscoll
- **Enlace**: https://python101.pythonlibrary.org/
- **Idioma**: Inglés
- **Descripción**: Libro completo con sección dedicada a clases y objetos.
