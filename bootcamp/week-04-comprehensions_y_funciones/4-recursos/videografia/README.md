### 💡 Tips para Ver Videos

1. **Velocidad**: Usa 1.25x o 1.5x si ya conoces lo básico
2. **Practica**: Pausa y escribe el código tú mismo
3. **Repite**: No temas ver el mismo video varias veces
4. **Notas**: Toma notas de conceptos clave
5. **Código**: Descarga el código de ejemplo si está disponible
