## 📖 Contenido por Cheat Sheet

### 1. Python Basics
- Variables y asignación
- Tipos de datos primitivos
- Operadores (aritméticos, comparación, lógicos)
- Strings y f-strings
- Input/Output
- Condicionales y match
- Bucles y comprensiones
- Funciones y lambdas
- Type hints
- Errores comunes

### 2. Data Structures
- Listas: métodos, slicing, ordenación
- Tuplas y namedtuple
- Diccionarios: operaciones, defaultdict, Counter
- Sets: operaciones de conjuntos
- Estructuras anidadas
- Comprensiones avanzadas
- Funciones built-in (map, filter, zip, enumerate)
- itertools y functools
- Patrones comunes
- Complejidad (Big O)

### 3. OOP Python
- Clases y objetos
- Atributos de clase vs instancia
- Métodos: instancia, clase, estático
- Propiedades (@property)
- Herencia simple y múltiple
- Mixins
- ABC y Protocol
- Métodos especiales (dunder)
- Dataclasses
- Patrones de diseño (Singleton, Factory, Builder, Observer)

### 4. File Handling
- Lectura y escritura de archivos
- Modos de apertura
- pathlib vs os.path
- JSON: load, dump, serialización
- CSV: reader, writer, DictReader
- Jerarquía de excepciones
- try/except/else/finally
- Excepciones personalizadas
- Context managers
- Operaciones de sistema (os, shutil, tempfile)

### 5. Testing & Debugging
- pytest: tests, assertions, markers
- Fixtures: scopes, setup/teardown, built-in
- Parametrización
- Mocking: Mock, patch, spy
- Cobertura con pytest-cov
- Debugging: print, breakpoint, pdb
- Logging: configuración, niveles, handlers
- Patrones de test (AAA, Given-When-Then)
- Test doubles (dummy, stub, spy, mock, fake)
- CLI y configuración de pytest
